# 高孟麟的網站專案

這是一個個人網站專案，展示了 HTML、CSS 和 JavaScript 的各種功能。

## 📁 專案結構

```
front-end/
├── index.html          # 主頁面（已重新設計，UI 精美）
├── css/                # CSS 樣式文件夾
│   ├── main.css       # 主要樣式表（現代化設計）
│   └── style.css      # 原有樣式表
├── html/               # 其他 HTML 頁面
│   ├── index2.html
│   ├── indexforcss.html
│   └── indexforcss2.html
├── assets/             # 媒體資源文件夾
│   ├── dog.jpg        # 圖片
│   ├── baby.mp3       # 音頻
│   └── cola.mp4       # 視頻
├── python/             # Python 腳本文件夾
│   ├── 0910.py
│   ├── 0911.py
│   ├── 0924hw.py
│   └── 1015.py
├── csspractice/        # CSS 練習文件
├── tests/              # 測試文件
└── README.md           # 專案說明文件
```

## ✨ 主要功能

- 🎨 **現代化 UI 設計**：採用漸變色背景、圓角卡片、陰影效果
- 📱 **響應式設計**：適配各種螢幕尺寸
- 🔗 **快速連結區**：卡片式佈局，展示常用連結
- 📝 **用戶註冊表單**：完整的表單驗證和美化樣式
- 🎯 **互動按鈕**：JavaScript 互動功能
- 🖼️ **嵌入內容**：iframe 展示外部網站
- 🎵 **媒體播放**：音頻、視頻、圖片展示
- 📊 **數據表格**：美化的營業時間表
- 📋 **列表展示**：有序、無序和描述列表
- ✨ **文字樣式**：各種文字格式示例

## 🚀 如何使用

1. 直接在瀏覽器中打開 `index.html` 文件
2. 或使用本地服務器運行專案
3. 探索各個功能模塊

## 🎨 設計特點

- 使用紫色漸變主題色 (#667eea → #764ba2)
- 卡片式佈局設計
- 平滑過渡動畫效果
- 懸停互動回饋
- 清晰的視覺層次
